Who should enroll in Web & Mobile Development?

People Looking For a New Career

Web Developers are well paid. According to Indeed, Web Developers make, on average, $60,000 a year in Utah. Most of our students leaving the program make about $16-$24 an hour, or between $33,000 to $49,000 a year upon graduation. Pay usually increases after about 1 year of experience.

Entrepreneurs or Freelancers

Whether you want to make a website for your company or you want to Freelance and make websites for others the Web & Mobile Development program can hlep you make your dream a reality. Many business owners don't have the budget to pay a Web Developer, the program will teach you how. As a Freelancer your income is only limited by how much you are willing to put into your business. Furthermore; there is nothing quite as rewarding as working on something you have built.

Current Developers Looking to Improve Their Skills

Are you currently employed as a junior or entry level developer and you want to improve your skills to get a better job? We can help you move beyond the basics so you can qualify for better employment.

Code Hobbyist

It doesn't matter if you are interested in blogging or just want to learn how to code. We can teach you how to develop an online presence and learn to code for fun! Many of us who code find writing code to be enjoyable and even therapeutic.

What will you learn?

HTML or HyperText Markup Language is used to structure websites.

CSS or Cascading Style Sheets are used to format the layout of webpages.

Databases are structured sets of data that can be accessed, managed, and updated.

PHP is an HTML embedded scripting language that can be used to access the server.

JavaScript is an object-oriented language used to create interactive effects on a webpage.

Mobile or Responsive Design is building websites to work on both desktops and mobile devices.

Our Instructors Are Out of This World!

All our our instructors have different specialties and interests. No matter what your interest is, or what development path you are interested in it's likely that one of our instructors can you help you succeed in your chosen path.

Front-End Development

Profile: front-end developers learn how to make the visual side of the web.

In the front-end track students will learn how to build and design the visual side of the web. Our front-end development track is particularly popular with people with the freelance spirit! In addition to the core skills you will learn advanced JS, WordPress and advanced CSS.

Back-End Development

Profile: post & retrieve data. Back-end developers work with data and servers.

Without Back-end developers the web is just a bunch of pretty pictures. Every time you buy something online or update your profile you are posting or retrieving data from a server and a back-end developer made it possible. Learn advanced back-end languages like Python, and advanced PHP as well as some of their frameworks.

JavaScript Developer

Profile: JS developers are true full stack developers.

Our JS track teaches you how to work with JavaScript and its various frameworks such as React, Angular and APIs. In the web development many people specialize in either front-end or back-end development, JS and its frameworks allow you to work on both the front and back end by using APIs to call and post data to the server.

UI/UX Design

Profile: design the web. Designers learn how to make websites that look good, are intuitive, and attract customers.

Web sites not only need to look great, but they also need to function properly and be easy and intuitive to use. Nothing is more frustrating to a user then not being able to figure out how to interact with an application. In the UI/UX track you will learn how to make websites that look great and are easy to use and help attract customers.
